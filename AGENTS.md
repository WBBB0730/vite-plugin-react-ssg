## 最高要求
- 不论用户用什么语言提问，都必须使用中文回答
- 只完成用户要求的任务，不擅自做任何多余的事情
- 做出任何不是用户显式要求的改动之前，必须先给出方案，询问用户是否接受
- 做出任何改动时，要有全局意识，考虑是否会影响到其他代码
- 如果有任何疑问，必须直接询问（最好使用询问工具），禁止做任何猜测
- 如果涉及到任何库/框架，必须先使用 context7 获取相关的文档（或直接访问官方文档），严格遵循官方推荐的最佳实践
- 在写代码/计划前，必须先阅读其他相似的代码，并保持一致的写法和风格
- 禁止在未经同意的情况下执行任何会导致 Git 状态变更的操作
- 严格遵循 SDD 原则，使用 openspec。
- 严格遵循 TDD 原则，使用 vitest
